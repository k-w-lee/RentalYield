## Factors you should not trust scraping alone for

| Factor                           | Why not enough                                        |
| -------------------------------- | ----------------------------------------------------- |
| **True transacted value**        | Asking price can be inflated. Need transaction data.  |
| **Actual rental achieved**       | Asking rent is not always actual signed rental.       |
| **Building management quality**  | Needs site visit, reviews, resident feedback.         |
| **Maintenance fee accuracy**     | Often missing or outdated in listings.                |
| **Vacancy risk**                 | Can estimate, but cannot confirm without market data. |
| **Flood/noise/traffic risk**     | Needs map check, site visit, local knowledge.         |
| **Legal/title issues**           | Need SPA/title/agent/lawyer confirmation.             |
| **Airbnb/short-stay permission** | Need JMB/MC rules, not just listing description.      |

## Best practical workflow

|  Step | Action                                                                                           |
| ----: | ------------------------------------------------------------------------------------------------ |
| **1** | Scrape PropertyGuru + iProperty listings.                                                        |
| **2** | Deduplicate by project name, unit size, price, bedroom, agent, and photos if possible.           |
| **3** | Calculate price psf, gross yield, estimated net yield, and cash flow.                            |
| **4** | Compare each listing against similar units in the same project.                                  |
| **5** | Shortlist top 10 by score.                                                                       |
| **6** | Manually verify maintenance fee, actual rent, transacted price, building quality, and occupancy. |
| **7** | View only the top 3–5 units physically.                                                          |

## My conclusion

Scraping is very strong for **shortlisting**, but weak for **final decision**.

Best use:

**Scraping = filter 100 listings into top 10**
**Manual research/site visit = choose the final 1**

The most reliable scraped shortlist formula:

**High rental yield + below-average price psf + strong MRT/job hub access + low competition + positive cash flow estimate.**
